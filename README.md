# University Dashboard System

## Overview

University Dashboard System is a web-based management application developed to simplify and manage university-related operations efficiently. The system allows administrators to manage universities, colleges, departments, staff members, students, and attendance records through a centralized dashboard.

The project was developed using PHP, MySQL, HTML, CSS, and JavaScript.

---

## Features

* University management
* College and department management
* Staff management system
* Student management system
* Attendance management
* Dynamic dropdown selection
* Photo upload functionality
* Responsive dashboard interface
